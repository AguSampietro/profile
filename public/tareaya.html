Personally

Technology Stack

With my limited experience (2016) in mobile/web development, I decided we should use MSSQL Server, C# (Rest API), and Android (with Java) for this project. Then, as my career in the mobile world progressed, I added more technological improvements, and today the app features the following technology stack:

  • MSSQL Server: This is still the main database where we store professional profiles.
  • C# Rest API: For the application logic and interaction with the database.
  • Firebase Auth: Handles users.
  • Firebase Firestore: We use it to store transactions and chats.
  • Firebase Messaging: Handles push notifications.
  • Firebase Functions: Transaction management and email sending.
  • Flutter for the Android and iOS apps, and also for the professional search website.

In conclusion, the app is still up and running. It's completely free for some South American countries. I hope to one day return to it with even more enthusiasm and be able to implement more of my ideas.
